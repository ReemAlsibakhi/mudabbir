## 🎯 الفكرة

مدبّر هو أول تطبيق عربي لإدارة المال الأسري مدعوم بالذكاء الاصطناعي، مصمم خصيصاً للثقافة العربية ويغطي **22 دولة عربية**.

## ✨ المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 🌙 **شاشة اليوم** | سجّل مصاريفك في 30 ثانية |
| 🎤 **إدخال صوتي** | "صرفت 50 ريال على الغداء" |
| 🔥 **سلسلة يومية** | نظام الاستمرارية والتحفيز |
| 🎯 **أهداف ذكية** | حاسبات تلقائية للادخار |
| 📊 **تقارير متقدمة** | مقارنة شهرية مفصّلة |
| 📅 **مصاريف ثابتة** | تُدخل مرة وتتكرر تلقائياً |
| 🦁 **شخصية الأسرة** | الأسد، النمر، الثعلب... |
| 🌙 **وضع رمضان** | ميزانية موسمية ذكية |

## 🏗️ البنية التقنية

```
مدبّر

```

---

## 📁 هيكل المشروع

```
lib/
├── core/
│   ├── constants/     # الثوابت، الثيم، الدول، الفئات
│   └── utils/         # مساعدات التاريخ والتحقق
├── data/
│   ├── models/        # نماذج البيانات (Hive)
│   └── providers/     # Riverpod Providers
├── features/
│   ├── onboarding/    # الإعداد الأولي
│   ├── daily/         # شاشة اليوم
│   ├── income/        # الدخل
│   ├── expenses/      # المصاريف
│   ├── goals/         # الأهداف
│   ├── reports/       # التقارير
│   ├── settings/      # الإعدادات
│   └── notifications/ # الإشعارات
└── shared/
    └── widgets/       # مكونات مشتركة
```

## 🚀 خطوات التشغيل

```bash
# 1. تثبيت الاعتماديات
flutter pub get

# 2. توليد الكود التلقائي (Hive + Riverpod)
dart run build_runner build --delete-conflicting-outputs

# 3. تشغيل التطبيق
flutter run
```
