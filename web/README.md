# مدبّر — Landing Page

موقع تعريفي بسيط للتطبيق.

## الملفات
- `index.html` — صفحة الهبوط الرئيسية
- `privacy.html` — سياسة الخصوصية (مطلوبة لـ Play Store)

## النشر
ارفعي هذين الملفين على:
- **Netlify**: اسحبي المجلد وأفلتيه على netlify.com/drop
- **GitHub Pages**: في إعدادات الريبو، فعّلي Pages من مجلد /web
- **Vercel**: نفس Netlify

## رابط سياسة الخصوصية لـ Play Store
`https://[رابطك]/privacy.html`
